A high-performance, responsive developer portfolio built with a terminal-inspired "HUD" aesthetic. This project showcases a clean MERN-stack focus, integrated project archives, and a mobile-optimized experience.

## 🚀 Live Demo

**View the live site here:** [kamosportfolio.netlify.app](https://kamosportfolio.netlify.app/)

---

## 🛠️ Technical Stack

- **Frontend:** React 18 + Vite
- **Styling:** Tailwind CSS (Fluid Typography & Custom Breakpoints)
- **Icons:** Lucide-React & React-Icons
- **Animations:** Framer Motion / Custom CSS Keyframes
- **Deployment:** Netlify

---

## ✨ Key Features

- **Adaptive HUD Layout:** A unique "Heads-Up Display" interface that adapts perfectly to desktop, tablet, and mobile browsers.
- **Terminal Typewriter Effect:** Dynamic role switching with a custom-coded typing animation.
- **Synchronized Breakpoints:** Custom responsive logic ensuring "Desktop Mode" stability on mobile devices.
- **Project Archive:** Categorized display of full-stack applications with direct links to Code and Demos.
- **Optimized Performance:** Built with Vite for lightning-fast HMR and optimized production bundles.

## ⚙️ Local Development

1. Clone the repository:

```bash
git clone git@github.com:KAMO333/Revamped-Portfolio.git
cd Revamped-Portfolio
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```
